## ğŸ“ Advanced: Full 45-Minute Lesson Plan

### Topic: "My Daily Routine" (Present Simple + Time Vocabulary)

**Materials Needed**: 
- Vocabulary Bank: Daily Routines category
- Grammar Bank: Present Simple
- Virtual Board with clock images

#### Warm-Up (5 min)
- Show routine vocabulary flashcards
- Students mime each action
- Practice pronunciation

#### Presentation (10 min)
1. Virtual Board:
   - Draw large clock
   - Add times around edge
   - Write routine next to each time:
     * 7:00 - Wake up
     * 7:30 - Eat breakfast
     * 8:00 - Go to school
     * etc.

2. Grammar Bank:
   - Review Present Simple rule
   - Show examples with routines
   - Emphasize "every day", "always", "usually"

#### Practice (15 min)
1. **Controlled Practice**:
   - Grammar exercises (fill-in-blank)
   - Error correction
   - Check answers together

2. **Guided Practice**:
   - Virtual Board: Draw your routine
   - Write 3 sentences
   - Share with partner

#### Production (10 min)
- Students create own routine board
- Draw 4 main activities
- Write sentences using Present Simple
- Present to class: "I wake up at..."

#### Review & Homework (5 min)
- Save student boards
- Review common mistakes
- Homework: 
  * Complete spelling practice
  * Interview family member about routine
  * Write 5 sentences
