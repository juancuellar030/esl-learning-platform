## Opening the Platform

### Method 1: Double-Click (Easiest)
1. Navigate to the `esl-platform` folder

1. Open your web browser (Chrome, Firefox, Safari, Edge)

1. Open your web browser
2. Press `Ctrl+O` (Windows) or `Cmd+O` (Mac)
3. Navigate to and select `index.html`
4. Click "Open"

## üéØ Common Tasks

### Create a Vocabulary Lesson
```
1. Go to Vocabulary Bank
2. Select 5-10 words
3. Click "Generate Exercise"
4. Choose exercise type
5. Press Ctrl+S to save
```

### Teach Grammar Live
```
1. Go to Grammar Bank
2. Select a rule (e.g., "Articles")
3. Review explanation with students
4. Generate exercises
5. Have students complete them
```

### Create a Visual Lesson
```
1. Go to Virtual Board
2. Select "Chalkboard" or "Nature" background
3. Use text tool to add title
4. Draw diagrams with pen
5. Add images if needed
6. Save your board
7. Export as PNG to share
```

### Make Flashcards for Homework
```
1. Go to Vocabulary Bank
2. Filter by category (e.g., "Animals")
3. Select all words in that category
4. Generate "Flashcards" exercise
5. Students can practice independently
```

## ‚ùì Quick Troubleshooting

**Q: The page won't load**
A: Make sure you're opening index.html in a web browser, not a text editor

**Q: My saved lessons disappeared**
A: They're stored in your browser. If you cleared browser data, they're gone. Export important boards as images!

**Q: Audio doesn't play**
A: Try using Chrome or Edge. Some browsers need permission for speech synthesis.

**Q: I can't draw on the board**
A: Check that the pen tool (‚úèÔ∏è) is selected and highlighted in blue

**Q: Colors don't change**
A: Click on a color in the color palette. The selected color has a gold border.

**Q: How do I add more words?**
A: Edit the js/data.js file - look for the vocabularyBank array
